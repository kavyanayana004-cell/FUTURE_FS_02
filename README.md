🍽️ Indian Aroma – Restaurant Website

📌 Project Overview

Indian Aroma is a modern and responsive restaurant website designed to showcase authentic Indian cuisine with a contemporary dining experience. The website provides visitors with information about the restaurant, menu offerings, customer testimonials, and an easy table reservation system.

Tagline: Where Tradition Meets Modern Taste

---

✨ Features

- Responsive and modern user interface
- Elegant hero section with call-to-action
- About Us section
- Menu showcase with food categories
- Customer testimonials
- Table reservation form
- Reservation cancellation option
- Contact information section
- Mobile-friendly design
- Smooth navigation and user experience

---

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts

🎯 Business Problem Solved

Many local restaurants struggle to establish a strong online presence. This website helps restaurants:

- Improve brand visibility
- Showcase menu offerings online
- Increase customer engagement
- Simplify table reservations
- Build customer trust through reviews
- Provide easy access to restaurant information

---

🚀 How to Run

1. Clone the repository

git clone <repository-url>

2. Open the project folder

cd Indian-Aroma

3. Open "index.html" in your browser
